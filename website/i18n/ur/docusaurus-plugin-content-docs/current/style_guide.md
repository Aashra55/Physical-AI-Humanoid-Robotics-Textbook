# فزیکل اے آئی اور ہیومنائڈ روبوٹکس ٹیکسٹ بک کے لیے مارک ڈاؤن اسٹائل گائیڈ

یہ اسٹائل گائیڈ مارک ڈاؤن میں لکھے گئے تمام کتابی مواد میں مستقل مزاجی اور پڑھنے کی اہلیت کو یقینی بناتا ہے۔

## سرخیاں

- تمام سرخیوں کے لیے ATX سرخیاں (جو `#` سے شروع ہوتی ہیں) استعمال کریں۔
- ہر صفحے پر صرف ایک H1 (`#`) (صفحے کا عنوان)۔
- سرخیاں ترتیب وار ہونی چاہئیں (مثال کے طور پر، H2 سے H4 پر نہ جائیں)۔

```markdown
# صفحے کا عنوان

## سیکشن کا عنوان

### سب سیکشن کا عنوان

#### مزید سب سیکشن
```

## پیراگراف اور لائن بریکس

- پیراگراف کو الگ کرنے کے لیے خالی لائنیں استعمال کریں۔
- جملوں کو مختصر رکھیں۔
- ٹیکسٹ ایڈیٹرز میں پڑھنے کی اہلیت کے لیے لائنوں کو تقریباً 80-100 حروف پر ہارڈ ریپ کریں۔

## زور

- *ترچھے حروف* کے لیے ``*سنگل اسٹارکس*`` استعمال کریں۔
- **بولڈ** کے لیے ``**ڈبل اسٹارکس**`` استعمال کریں۔
- ***بولڈ ترچھے حروف*** کے لیے ``***ٹرپل اسٹارکس***`` استعمال کریں۔

## فہرستیں

- غیر ترتیب شدہ فہرستوں کے لیے ہائفنز (`-`) استعمال کریں۔
- ترتیب شدہ فہرستوں کے لیے `1.` استعمال کریں۔ اصل نمبر اہمیت نہیں رکھتا؛ مارک ڈاؤن اسے صحیح طریقے سے پیش کرے گا۔
- گھونسلا شدہ فہرستوں کو دو خالی جگہوں کے ساتھ انڈنٹ کریں۔

```markdown
- آئٹم 1
  - گھونسلا شدہ آئٹم 1.1
    - گھونسلا شدہ آئٹم 1.1.1
- آئٹم 2

1. ترتیب شدہ آئٹم 1
  1. گھونسلا شدہ ترتیب شدہ آئٹم 1.1
2. ترتیب شدہ آئٹم 2
```

## کوڈ

- `ان لائن کوڈ` کے لیے بیک ٹکس (`` `کوڈ` ``) استعمال کریں۔
- کوڈ کے بلاکس کے لیے زبان کی وضاحت کرنے والوں کے ساتھ فینسڈ کوڈ بلاکس استعمال کریں۔
- نحو کو نمایاں کرنے کے لیے ہمیشہ زبان کی وضاحت کریں۔

```python
def hello_world():
    print("Hello, Physical AI!")
```

## لنکس

- جب بھی ممکن ہو ان لائن لنکس استعمال کریں۔
- وضاحتی لنک ٹیکسٹ فراہم کریں۔

```markdown
[ڈوکوسارس ویب سائٹ](https://docusaurus.io/)
```

## تصاویر

- تصاویر کو ڈوکوسارس پروجیکٹ کے اندر `static/img` فولڈر میں اسٹور کریں۔
- وضاحتی Alt ٹیکسٹ استعمال کریں۔

```markdown
![تصویر کے لیے Alt ٹیکسٹ](/img/my-diagram.png)
```

## میزیں

- جدا کاروں کے لیے ہائفنز کی مستقل تعداد استعمال کریں۔
- کالونز کا استعمال کرتے ہوئے کالموں کو سیدھ میں لائیں۔

```markdown
| ہیڈر 1 | ہیڈر 2 | ہیڈر 3 |
| :------- | :------- | :------- |
| مواد  | مواد  | مواد  |
```

## بلاک کوٹس

- بلاک کوٹس کے لیے `>` استعمال کریں۔

```markdown
> "اے آئی کا مستقبل جسمانی ہے۔" - غیر منسوب
```

## اڈمونیشنز (کال آؤٹس)

- نوٹس، انتباہات وغیرہ کے لیے ڈوکوسارس اڈمونیشن نحو استعمال کریں۔

```markdown
:::note
یہ ایک نوٹ ہے۔
:::

:::warning
یہ ایک انتباہ ہے۔
:::
```

## تبصرے

- ایسے تبصروں کے لیے HTML تبصرے استعمال کریں جو پیش نہیں کیے جانے چاہئیں۔

```html
<!-- یہ ایک تبصرہ ہے -->
```
