# ٹیوٹوریل کا تعارف

آئیے **5 منٹ سے بھی کم وقت میں ڈوکوسارس کو دریافت کریں**۔

## شروع کرنا

**ایک نئی سائٹ بنا کر** شروع کریں۔

یا **[docusaurus.new](https://docusaurus.new)** کے ساتھ **فوری طور پر ڈوکوسارس آزمائیں**۔

### آپ کو کیا ضرورت ہو گی

- [Node.js](https://nodejs.org/en/download/) ورژن 20.0 یا اس سے اوپر:
  - Node.js انسٹال کرتے وقت، آپ کو تجویز دی جاتی ہے کہ انحصار سے متعلق تمام چیک باکسز کو چیک کریں۔

## ایک نئی سائٹ بنائیں

**کلاسک ٹیمپلیٹ** کا استعمال کرتے ہوئے ایک نئی ڈوکوسارس سائٹ بنائیں۔

کمانڈ چلانے کے بعد کلاسک ٹیمپلیٹ خود بخود آپ کے پروجیکٹ میں شامل ہو جائے گا:

```bash
npm init docusaurus@latest my-website classic
```

آپ اس کمانڈ کو کمانڈ پرامپٹ، پاور شیل، ٹرمینل، یا اپنے کوڈ ایڈیٹر کے کسی دوسرے مربوط ٹرمینل میں ٹائپ کر سکتے ہیں۔

کمانڈ ڈوکوسارس کو چلانے کے لیے درکار تمام ضروری انحصار کو بھی انسٹال کرتی ہے۔

## اپنی سائٹ شروع کریں

ڈویلپمنٹ سرور چلائیں:

```bash
cd my-website
npm run start
```

`cd` کمانڈ اس ڈائرکٹری کو تبدیل کرتی ہے جس کے ساتھ آپ کام کر رہے ہیں۔ اپنی نئی بنائی ہوئی ڈوکوسارس سائٹ کے ساتھ کام کرنے کے لیے، آپ کو ٹرمینل کو وہاں نیویگیٹ کرنے کی ضرورت ہوگی۔

`npm run start` کمانڈ آپ کی ویب سائٹ کو مقامی طور پر بناتی ہے اور اسے ڈیولپمنٹ سرور کے ذریعے پیش کرتی ہے، جسے آپ http://localhost:3000/ پر دیکھنے کے لیے تیار ہیں۔

`docs/intro.md` (یہ صفحہ) کھولیں اور کچھ لائنیں ترمیم کریں: سائٹ **خود بخود دوبارہ لوڈ ہو جاتی ہے** اور آپ کی تبدیلیاں دکھاتی ہے۔
